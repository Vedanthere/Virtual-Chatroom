Features 🌟

1.   This web page adapts a new type of custom cursor for user to navigates the site.

2.   /users : lists of all users in the room.

3.   Users will be able to create new rooms with custom room names.

4.   Users will also be able to join existing rooms by typing in the room code in the text box provided.

5.   Tic Tac Toe : User can play an interactive game of tic tac toe.

6.   /testing : opens a new link which contains all the rooms in the order were created.

7.   Whiteboard : User can open whiteboard to draw (color and stroke with customization).
     The whiteboard also has a customized cursor for navigating the tool bar, but has a crosshair for the drawing board to allow for more precise strokes.

8.   Task List : Sleek and user-friendly interface to create and manage tasks.

9.   Memes (a fun and chill game to relieve stress) : User can play number guessing with the computer.

10.   About Page : Shows the participation of the group and shines a light on additional features.
